# リポジトリ

## GitHub公式
- https://github.com/github
- 一見した感じ「-」区切りが多い印象
- 一文字目は基本小文字
- 「VSCode」などのリポジトリはあったが例外的

## 富士通

## まとめ
- リポジトリ名は、小文字始まりの「-」つなぎで命名するのが一般的のよう
- その他慣例的なものとしてこんなのがあるよというのが以下のサイト
